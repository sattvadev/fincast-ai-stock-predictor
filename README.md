# FinCast: AI Stock Predictor

[![Deploy to Cloudflare](https://deploy.workers.cloudflare.com/button)](https://deploy.workers.cloudflare.com/?url=https://github.com/sattvadev/fincast-ai-stock-predictor)

> A minimalist and visually stunning web interface to predict and visualize future stock prices using an AI-powered API.

FinCast is a visually stunning, minimalist web application designed to provide a seamless interface for an underlying stock prediction API. The application focuses on a clean, intuitive user experience, allowing users to input a stock ticker, select a prediction timeframe, and instantly visualize the predicted stock prices on an interactive chart. The design prioritizes clarity, performance, and aesthetic elegance, making financial data accessible and beautiful.

## ‚ú® Key Features

-   **Intuitive Interface**: A clean, single-page dashboard for a seamless user experience.
-   **Dynamic Predictions**: Input any stock ticker and select a future timeframe for prediction.
-   **Interactive Charting**: Visualize historical and predicted stock data with beautiful, responsive charts powered by Recharts.
-   **Elegant UI**: Built with shadcn/ui and Tailwind CSS for a modern, polished look.
-   **Polished User Experience**: Smooth animations, elegant loading states, and clear feedback for all interactions.
-   **Responsive Design**: Flawless performance and layout across all device sizes, from mobile to desktop.

-   **Frontend**: React, TypeScript, Vite

## üèÅ Getting Started

Follow these instructions to get a local copy up and running for development and testing purposes.

### Prerequisites

-   [Node.js](https://nodejs.org/) (v18 or later)
-   [Bun](https://bun.sh/)
-   [Git](https://git-scm.com/)

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/your-username/fincast_stock_predictor.git
    cd fincast_stock_predictor
    ```

2.  **Install dependencies:**
    This project uses Bun as the package manager.
    ```sh
    bun install
    ```

### Running the Development Server

To start the local development server, which includes the Vite frontend and the Hono backend worker, run:

```sh
bun run dev
```

The application will be available at `http://localhost:3000`.

## üèóÔ∏è Project Structure

-   `src/`: Contains all the frontend React application code, including pages, components, hooks, and styles.
-   `worker/`: Contains the Cloudflare Worker backend code, built with Hono. This is where API routes are defined.
-   `shared/`: Contains shared TypeScript types used by both the frontend and the backend to ensure type safety.

## üîß Development

-   **Frontend**: All frontend components are located in `src/components`. The main application view is `src/pages/HomePage.tsx`.
-   **Backend**: API endpoints are defined in `worker/user-routes.ts`. You can add or modify routes here.
-   **Styling**: The project uses Tailwind CSS for styling. You can customize the theme in `tailwind.config.js` and global styles in `src/index.css`.

## ‚òÅÔ∏è Deployment

This project is configured for easy deployment to Cloudflare Workers.

1.  **Login to Cloudflare:**
    If you haven't already, log in to your Cloudflare account via the command line:
    ```sh
    bunx wrangler login
    ```

2.  **Deploy the application:**
    Run the deploy script, which will build the application and deploy it to your Cloudflare account.
    ```sh
    bun run deploy
    ```

Wrangler will handle the entire process of building the frontend assets, bundling the worker, and deploying them.

Alternatively, you can deploy directly from your GitHub repository with one click:
